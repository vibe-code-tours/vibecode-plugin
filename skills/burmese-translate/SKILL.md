# Burmese Translation (EN + မြန်မာ)

For community-facing content, ship English and Burmese stacked.

## Rules
- Keep **technical tokens in English**: file paths, commands, `.mcp.json`, `git`, API names, code.
- Translate the **explanation** into plain, everyday Burmese — not formal/literary. Write like you'd speak to a friend.
- Keep numbers and ⭐/✅ emoji as-is.
- Mirror structure: same headings/steps in both languages so readers can match them.

## Pattern
```
📢 <English message>

──────────

📢 <same message in Burmese, technical terms left in English>
```

When unsure of a term, leave it in English rather than inventing a Burmese word nobody uses.
